# MiniMax 音乐批量生成工具 (MiniMax Music Batch Generator)

这是一个专为 MiniMax 音乐创作页面设计的油猴 (Tampermonkey) 脚本。它能够实现提示词的自动化批量输入与串行生成，并具备完善的防检测与异常处理机制。

## 🌟 核心功能

- **自动化串行生成**：自动按顺序输入提示词，监控生成状态，上一条完成后自动开始下一条。
- **三阶段深度检测**：
  1. **作品出现检测**：监控作品列表变化。
  2. **生成进度检测**：识别作品卡片加载状态。
  3. **按钮就绪检测**：确保系统后台完全处理完毕且生成按钮恢复可用后再继续。
- **拟人化防检测**：在每条任务之间增加 **5-10秒** 的随机随机观察间隔，模拟真人操作。
- **超长柔性等待**：支持长达 **5分钟** 的按钮就绪等待，自动应对系统繁忙或长歌曲生成。
- **虚拟列表优化**：自动执行列表置顶滚动，消除由于页面虚拟滚动导致的检测误判。
- **可视化悬浮面板**：支持拖拽、折叠，实时显示进度、倒计时及详细状态。
- **详尽诊断日志**：所有操作逻辑均同步至浏览器控制台 (F12)，方便排查异常。

## 🚀 安装与使用

1. **安装环境**：确保您的浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. **添加脚本**：点击 Tampermonkey 插件图标 -> "添加新脚本"，将 `minimax-music-batch.user.js` 的内容全选覆盖进去并保存。
3. **访问页面**：打开 [MiniMax 音乐创作](https://www.minimaxi.com/audio/music)。
4. **批量操作**：
   - 在左下角出现的悬浮面板中输入批量提示词（每行一条）。
   - 点击 **“开始批量”**。
   - 脚本将自动接管页面执行。

## ⚠️ 注意事项

- 建议单次批量生成的数量控制在 30 条以内，以确保账号安全。
- 在脚本运行期间，请尽量保持页面位于前台标签页，建议不要在生成过程中手动修改提示词输入框。
- 如遇“已停止”报错，请按 **F12** 查看控制台日志中 `[MiniMaxBatch]` 的详细报错信息。
